## 📖 Descrição
- Projeto Full Stack, utilizando Java Spring, Angular e PostgreSQL.
- Gerenciador de Finanças com Gráfico.
- REST API
- WebSite
- Interface para criar, alterar, deletar e visualizar finanças.
- Extras: TailwinCSS, FontAwsome, Chart.js
